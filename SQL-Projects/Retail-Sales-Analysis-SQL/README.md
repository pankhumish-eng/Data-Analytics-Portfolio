# Retail Sales Analysis using SQL

## Project Overview

This project analyzes retail sales data using SQL to answer business questions and generate actionable insights.

## Business Questions

1. Which region generates the highest sales?
2. Which category generates the most revenue?
3. Which states contribute the most sales?
4. What are the top-selling products?
5. Which products generate the highest profit?

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- Aggregate Functions
- CASE Statements

## Tools

- SQL
- Excel Dataset
- GitHub

## Expected Outcome

Business insights from retail sales data using SQL queries.
