# Retail Sales Dashboard

## Project Objective

Analyze retail sales data to identify sales trends, top-performing products, customer behavior, and regional performance.

## Business Questions

1. Which region generates the highest revenue?
2. Which products sell the most?
3. What is the monthly sales trend?
4. Who are the top customers?
5. Which category generates the most profit?

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Dashboard Design

## Key Metrics

- Revenue
- Profit
- Orders
- Customers
- Sales by Region
