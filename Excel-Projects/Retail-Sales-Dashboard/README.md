# Retail Sales Dashboard

## Project Objective

Analyze retail sales data to identify sales trends, top-performing products, customer behavior, and regional performance.

## Business Questions

1. Which region generates the highest revenue?
2. Which products sell the most?
3. What is the monthly sales trend?
4. Who are the top customers?
5. Which category generates the most profit?

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Dashboard Design

## Key Metrics

- Revenue
- Profit
- Orders
- Customers
- Sales by Region

  ## Key Findings

- Total Sales: 2,297,201
- Total Profit: 286,397
- Total Quantity Sold: 37,873
- West region generated the highest sales.
- Technology was the top-performing category.
- California was the highest revenue-generating state.
